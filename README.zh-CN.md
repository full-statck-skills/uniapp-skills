## 📖 简介

**uni-app 技能** 是一组 AI 编码智能体技能，属于 [Full Stack Skills](https://github.com/partme-ai/full-stack-skills) 生态，由 [PartMe.AI](https://github.com/partme-ai) 维护。包含 **13 个技能**。

## 📦 安装

```bash
npx skills add full-statck-skills/uniapp-skills
```

### Claude Code 安装

**方式一：npx skills CLI（推荐）**

```bash
npx skills add full-statck-skills/uniapp-skills
```

**方式二：手动安装**

```bash
git clone https://github.com/full-statck-skills/uniapp-skills.git
cp -r uniapp-skills/skills/* .claude/skills/
```

更多详情请参阅 [Claude Code 技能指南](https://code.claude.com/docs/en/skills) 和 [Agent Skills 规范](https://agentskills.io/)。
